Download & Installation

To Download Iguana you have currently two options.

  • Download the release files (jars)
  • Download the Source Code

    • Release Files

    Core Controller

    Download

    Result Processor

    Download

    To execute the core controller use the following line. (The Core controller is a daemon process and the actual benchmark will be send later on) ~$ java -Xmx4g -jar iguana.corecontroller-VERSION.jar org.dice_group.iguana.cc.MainController &

    To execute the result processor use the following line. (The Result Processor is a daemon process) ~$ java -Xmx4g -jar iguana.resultprocessor-VERSION.jar org.dice_group.iguana.rp.MainController &

    Source Code

    You can Download the Source Code by cloning the repository of GitHub with the following command:

    ~$ git clone -b master --single-branch https://github.com/AKSW/IGUANA.git

    Use maven to create the distribution

    ~$ cd IGUANA
    ~$ mvn clean package -Dmaven.test.skip=true
